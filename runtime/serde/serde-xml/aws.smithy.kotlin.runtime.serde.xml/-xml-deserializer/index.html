Parameters

reader

underlying XmlStreamReader from which tokens are read

validateRootElement

Flag indicating if the root XML document XmlToken.BeginElement should be validated against the descriptor passed to deserializeStruct. This only affects the root element, not nested struct elements. Some restXml based services DO NOT always send documents with a root element name that matches the shape ID name (S3 in particular). This means there is nothing in the model that gives you enough information to validate the tag.
